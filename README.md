# 🚩 FlagForge

FlagForge is a full-stack feature flag management platform that helps developers enable, disable, and manage application features without redeploying their applications.

## Features

- User Authentication
- Create, Update and Delete Feature Flags
- Enable/Disable Flags
- REST API
- MongoDB Database
- Docker Support
- CI/CD Ready

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Mongoose
- Docker
- GitHub Actions

## Getting Started

```bash
git clone https://github.com/your-username/FlagForge.git
cd FlagForge
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Using Docker

```bash
docker compose up --build
```
